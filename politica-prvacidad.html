2. ¿Qué Datos Personales Recopilamos?

Dependiendo de tu interacción con nuestros servicios, podemos recopilar los siguientes tipos de datos:

2.1. Datos de Identificación

  • Nombre y apellidos
  • Razón social (para empresas)
  • NIF/CIF/NIE
  • Dirección postal completa
  • Teléfono de contacto
  • Correo electrónico

2.2. Datos de Navegación

  • Dirección IP
  • Tipo de navegador y dispositivo
  • Páginas visitadas y tiempo de permanencia
  • Cookies y tecnologías similares (ver sección 7)

2.3. Datos de Contratación

  • Plan de hosting contratado
  • Método de pago utilizado
  • Historial de transacciones
  • Datos de facturación

2.4. Datos de Comunicación

  • Consultas enviadas a través de formularios de contacto
  • Conversaciones con soporte técnico
  • Correos electrónicos intercambiados

3. ¿Para Qué Usamos Tus Datos?

Utilizamos tus datos personales para las siguientes finalidades:

  1. Prestación del Servicio: Gestionar tu cuenta de hosting, procesar pagos, y proporcionar soporte técnico.
  2. Cumplimiento Legal: Emitir facturas, cumplir con obligaciones fiscales y legales.
  3. Comunicaciones Comerciales: Enviarte información sobre actualizaciones, nuevos servicios o promociones (solo si has dado tu consentimiento).
  4. Mejora del Servicio: Analizar el uso de nuestro sitio web para mejorar la experiencia del usuario.
  5. Seguridad: Detectar y prevenir fraudes, abusos o actividades maliciosas.
  6. Atención al Cliente: Responder a consultas, solicitudes y reclamaciones.

4. Base Legal para el Tratamiento

El tratamiento de tus datos se basa en:

  • Ejecución de un Contrato: Necesitamos tus datos para prestarte el servicio de hosting que has contratado.
  • Obligación Legal: Debemos conservar ciertos datos para cumplir con la legislación fiscal y mercantil.
  • Consentimiento: Para enviarte comunicaciones comerciales, solo si has aceptado explícitamente recibirlas.
  • Interés Legítimo: Para mejorar nuestros servicios y garantizar la seguridad de nuestros sistemas.

5. ¿Con Quién Compartimos Tus Datos?

Tus datos personales no se venden ni se ceden a terceros con fines comerciales. Solo compartimos información cuando es estrictamente necesario:

5.1. Proveedores de Servicios

  • Procesadores de Pago: Para procesar transacciones de forma segura
  • Proveedores de Hosting: Servidores donde alojamos nuestros servicios
  • Servicios de Email: Para enviar comunicaciones transaccionales
  • Herramientas de Análisis: Google Analytics para mejorar la experiencia del usuario

5.2. Autoridades Competentes

Podemos compartir datos cuando sea requerido por ley, orden judicial o autoridad competente.

Nota Importante: Todos nuestros proveedores están ubicados en la Unión Europea o cumplen con acuerdos de transferencia internacional de datos conforme al RGPD.

6. Tus Derechos Sobre Tus Datos

Como usuario, tienes los siguientes derechos reconocidos por el RGPD:

Derecho de Acceso

Puedes solicitar una copia de todos los datos personales que tenemos sobre ti.

Derecho de Rectificación

Puedes solicitar la corrección de datos inexactos o incompletos.

Derecho de Supresión

Puedes solicitar la eliminación de tus datos cuando ya no sean necesarios.

Derecho de Oposición

Puedes oponerte al tratamiento de tus datos para fines de marketing directo.

Derecho de Limitación

Puedes solicitar que limitemos el tratamiento de tus datos en ciertos casos.

Derecho de Portabilidad

Puedes solicitar recibir tus datos en un formato estructurado y transferirlos a otro proveedor.

¿Cómo Ejercer Tus Derechos?

Para ejercer cualquiera de estos derechos, envía un correo electrónico a ventas@esenex.es indicando:

  • Tu nombre completo y documento de identidad
  • El derecho que deseas ejercer
  • Dirección de correo electrónico para respuestas

Responderemos a tu solicitud en un plazo máximo de 30 días.

7. Uso de Cookies

Nuestro sitio web utiliza cookies y tecnologías similares para mejorar tu experiencia de navegación.

7.1. ¿Qué son las Cookies?

Las cookies son pequeños archivos de texto que se almacenan en tu dispositivo cuando visitas un sitio web. Permiten que el sitio recuerde tus acciones y preferencias durante un período de tiempo.

7.2. Tipos de Cookies que Utilizamos

  • Cookies Técnicas (Necesarias): Esenciales para el funcionamiento del sitio. No requieren consentimiento.
  • Cookies Analíticas: Nos ayudan a entender cómo los usuarios interactúan con nuestro sitio (Google Analytics).
  • Cookies de Preferencias: Recuerdan tus preferencias de idioma, modo oscuro, etc.

7.3. Gestión de Cookies

Puedes configurar tu navegador para rechazar todas las cookies o para que te avise cuando se envía una cookie. Sin embargo, algunas funciones del sitio pueden no funcionar correctamente si deshabilitas las cookies.

Cómo desactivar cookies:

  • Google Chrome
  • Mozilla Firefox
  • Safari
  • Microsoft Edge

8. Seguridad de Tus Datos

Implementamos medidas técnicas y organizativas apropiadas para proteger tus datos personales contra:

  • Acceso no autorizado
  • Pérdida o destrucción accidental
  • Uso indebido o alteración
  • Divulgación no autorizada

Medidas de Seguridad Implementadas:

  • Certificados SSL/TLS para cifrado de comunicaciones
  • Firewalls y sistemas de detección de intrusos
  • Copias de seguridad periódicas
  • Acceso restringido a datos personales solo al personal autorizado
  • Contraseñas seguras y políticas de acceso estrictas

9. ¿Cuánto Tiempo Conservamos Tus Datos?

Conservamos tus datos personales solo durante el tiempo necesario para cumplir con las finalidades para las que fueron recopilados:

  • Datos de clientes activos: Mientras mantengas una cuenta activa con nosotros
  • Datos de facturación: 6 años (según legislación fiscal española)
  • Datos de marketing: Hasta que retires tu consentimiento
  • Datos de navegación: 24 meses (Google Analytics)

Una vez transcurridos estos plazos, procederemos a la eliminación segura de tus datos.
